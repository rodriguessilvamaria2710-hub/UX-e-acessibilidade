# UX e Acessibilidade

Projeto de interface inclusiva focado em segurança digital para idosos.

Este site apresenta:

- navegação clara com links de salto e elementos semânticos;
- contraste de cores acessível e modo de alto contraste;
- controle de tamanho de texto para melhorar a leitura;
- formulário simples com feedback acessível por leitor de tela;
- foco visível e navegação por teclado.

Use `index.html` para ver a página, `style.css` para o estilo e `script.js` para as interações de acessibilidade.
